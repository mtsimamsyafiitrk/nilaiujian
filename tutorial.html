3
Atur Pengaturan Nilai (Opsional)
Sesuaikan jumlah tugas, UH, dan bobot penilaian sesuai mapel Anda
💡
Kapan perlu diubah?

Hanya perlu diubah jika mapel Anda memiliki jumlah tugas/UH yang berbeda dari default, atau bobot formatif/sumatif yang berbeda. Jika tidak diubah, sistem menggunakan pengaturan default dari admin.

a
Klik tombol ⚙️ di pojok kanan atas

Tombol ini ada di halaman input nilai. Modal pengaturan akan terbuka.

b
Sesuaikan pengaturan

Ada 4 hal yang bisa diatur:

⚙️ Pengaturan Nilai ✕
ℹ️ Default global: 6 tugas, 7 UH, UTS: Ya, Bobot 60%/40%
Jumlah Tugas
Jumlah UH
✓
Ada UTS
Bobot Formatif: 60% — Sumatif: 40%

Penjelasan setiap pengaturan:

Jumlah Tugas Berapa kolom tugas (formatif) yang tersedia. Misal: 2 tugas → kolom Tugas 1 dan Tugas 2.
Jumlah UH Berapa Ulangan Harian (sumatif) dalam satu semester.
Ada UTS Centang jika mapel Anda mengadakan Ujian Tengah Semester.
Bobot Formatif/Sumatif Geser slider untuk mengatur persentase. Default: Formatif 60% + Sumatif 40% = 100%.
c
Klik "Simpan & Terapkan"

Tabel nilai akan langsung diperbarui sesuai pengaturan baru. Pengaturan ini hanya berlaku untuk mapel yang sedang dibuka.

c
Cara mengisi nilai

Klik kotak input pada baris siswa, lalu ketik angka nilainya (0–100). Nilai Rapor, Huruf, Ijazah, dan Deskripsi akan langsung terhitung otomatis.

d
Kolom yang tidak diisi dianggap 0

Jika ada kolom yang dikosongkan (tidak diketik apapun), sistem otomatis menganggapnya sebagai nilai 0 dalam perhitungan rata-rata. Pastikan mengisi semua kolom yang relevan.

e
Kolom Nilai UM (Ujian Madrasah)

Kolom ini berwarna ungu muda. Isi setelah nilai ujian madrasah keluar. Jika belum ada, boleh dikosongkan — Nilai Ijazah sementara akan sama dengan Nilai Rapor.

Rumus Perhitungan Otomatis
// Nilai Formatif (A)
A = Rata-rata semua Tugas (kolom kosong dihitung 0)

// Nilai Sumatif (B)
B = Rata-rata semua UH + UTS (kolom kosong dihitung 0)

// Nilai Rapor
Nilai Rapor = (A × Bobot Formatif% + B × Bobot Sumatif%) ÷ 100

// Nilai Ijazah (setelah UM diisi)
Nilai Ijazah = Nilai Rapor × 60% + Nilai UM × 40%

6
Simpan Nilai
Pastikan semua nilai sudah benar sebelum menyimpan
a
Periksa kembali semua nilai

Gulir tabel dari atas ke bawah, pastikan tidak ada siswa yang terlewat dan nilai sudah benar.

b
Klik tombol 💾 Simpan di pojok kanan atas

Tombol ini ada di topbar halaman input nilai.

c
Tunggu konfirmasi berhasil

Tombol akan berubah menjadi "⏳ Menyimpan..." saat proses berlangsung, lalu muncul notifikasi hijau jika berhasil tersimpan.

⚠️
Penting: Simpan per kelas

Setiap kali berpindah kelas (misal dari IX-A ke IX-B), isi nilai dulu untuk kelas tersebut lalu klik Simpan sebelum pindah kelas. Nilai yang belum disimpan akan hilang jika langsung pindah.

✅
Data tersimpan otomatis ke server

Nilai yang sudah disimpan langsung masuk ke database dan bisa dilihat admin madrasah. Anda bisa kembali lagi kapan saja untuk mengedit jika ada koreksi.

7
Pindah Kelas & Selesai
Ulangi proses untuk kelas lain, lalu logout dengan aman
a
Untuk mengisi kelas lain

Simpan nilai kelas saat ini terlebih dahulu, kemudian klik tab kelas lain (misal: IX-B). Ulangi langkah 5 dan 6 untuk kelas berikutnya.

IX-A sudah tersimpan, sekarang mengisi IX-B
b
Untuk mengisi mapel lain

Klik tombol ◀ di pojok kiri atas untuk kembali ke dashboard, lalu pilih mapel berikutnya.

c
Logout setelah selesai

Klik ikon 🚪 di pojok kanan atas topbar, lalu konfirmasi logout. Selalu logout terutama jika menggunakan perangkat bersama.

💡
Nilai bisa diedit kapan saja

Login kembali → pilih mapel yang sama → data nilai yang sudah tersimpan akan muncul kembali dan bisa diubah. Klik Simpan lagi setelah ada perubahan.
